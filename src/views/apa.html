Remote Diagnostics i Operational Analysis Web

  • För användare med behörighet finns nu möjlighet att beställa nya utläsningar för valt fordon eller vald motor genom att klicka på den nya knappen Beställ ny utläsning i produktvalspanelen.
  • För användare med behörighet finns nu möjlighet att se sina utläsningsbeställningar under Mina Beställningar i menyn.
  • Denna funktionalitet kommer att ersätta den nuvarande applikationen Remote Diagnostics Tool.
